# Фан сайт присвячений серіалу Майстри меча онлайн(Sword Art Online).
- **fancybox** використовується для створення плеєру для програвання серії на сторінці;
- **slick** використовується для створення каруселі слайдів, де створено опис героїв.

Сайт будується за допомогою **Webpack** 

- папка **src** потрібна для пободування шляху за для роботи **webpack**.
